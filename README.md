# Nest and Prisma app

## Description

A simple project to test how [Prisma](https://www.prisma.io/) can be used within [Nest](https://nestjs.com/) app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
